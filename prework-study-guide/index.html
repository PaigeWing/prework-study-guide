HTML

  • There are many elements, but the two most important elements are the head and body elements
  • The head element contains information about the webpage
  • Head elements are not visible to the user
  • The head element acts as the container for the metadata
  • The body element contains the content that is shown to the user
  • The body is also known as the parent element
  • The elements nested within the body element are known as the child elements

Git

  • git status: checks what branch we are currently on
  • git checkout branch name: takes you to that branch if already created
  • git checkout -b branch name: creates a new branch
  • git add A-: adds modifications for the current working branch to the staging area and commits them
  • git pull origin main: pulls the most current version of the main branch from the repository to your local computer, which should be done before starting new work
  • git push origin branch name: pushes the new branch that you are working on locally to the remote GitHub
